CSS 
CSS stands for Cascading Style Sheet.
===================================

To apply look and feel for our web pages we using css.

The main advantage of css is 
1)improve the reusability.
2)reduce the file size 
3)improve the band width.


How we can develop css classes
===============================

we can develop classes in following ways.
1)External css. in that case file extention must be .css
2)internal css;
    These css will be written in side of <head> tag, 
    by using <style>
3)inline css :
    if we apply css properties using style attribute of element
    then those are called as in line css.
    

1)External css
================
If we want to integrate external css we can do in various ways.
1)By using link tag.
link tag is used in side of head tag.

2)@import 
@import is used in some other .css file;

what are the box modelling properties?

padding: 10px 20px 30px 40px; [t, r, b, l]
padding: 20px 50px; [(t,b), (r, l)]
padding: 10px 50px 40px ; [t, (r,l), b];




margin: space
 margin is nothing but outter space of element (out side of border).

border: border is nothing border;

padding: space
    padding is nothing but inner space between border and content.

CSS Selectors:
==============
What is the selector?
===================
The way of writing or defining css classes and making link with element 
is called as selector

1)Simple Selectors
    1)element name based selectors
    2)class based selectors
    3)id selector;

1)element name based selectors
==============================
1)takes element name it self to define classes  
2)In this case in entire document, where ever thse html elements are there ,
for those elements these select styles will be applied directly.
3)you no need to link with elements manually


body{

}
ul{

} 
li{

}

2)class based selectors
========================
1)for clas based select, you can choose any name, usually business contex related names.
2)select must start (.)
3)how to link with elements
    Thse class based selectors will linked with elements using "class" attribute of element.

 3)id selector
 =============
 1)thse selector will be defined by using id names of elements.
 2)these selelctors must start with (#)
 3)How to apply to elements?
    Not need to apply or make link with elements manually.
    It is directly applied, if element holds id
    <style>
        #id1{
                
        }
    </style>
    <div id="id1">

    </div>
